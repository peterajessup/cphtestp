IBM SDK for Linux, Java Technology Edition, Versão 8
=====================================================

Este arquivo LEIA-ME se aplica à versão 8, liberação 0, modificação 0, fix pack 0 (VRMF
8.0.0.0) e a todas as liberações, modificações e fix packs subsequentes, até que seja
indicado de outra maneira em um novo arquivo LEIA-ME.

A IBM fornece conteúdo adicional com o SDK.
